📋 탭 1 — 매출 현황

1
채널 추가
상단의 채널 추가 영역에서 판매 중인 쇼핑몰을 선택하고 "+ 채널 추가" 버튼을 클릭합니다. 여러 채널을 한 번에 추가하려면 "전체 채널 일괄 추가"를 사용하세요.
💡 목록에 없는 채널은 "직접 입력..."을 선택하면 입력창이 나타납니다.
2
주차 추가
날짜를 선택하면 해당 날짜가 포함된 주(월~일)가 자동 계산됩니다. 오른쪽에서 특정 채널 또는 전체 채널을 선택한 후 "+ 주차 추가"를 클릭하세요.
3
실매출액 입력
두 가지 방법이 있습니다:
방법 A (자동) — 채널명 옆 📤 버튼을 클릭해 쇼핑몰 정산/주문 엑셀 파일을 업로드하면 날짜·금액 컬럼을 자동 인식해 주 단위로 집계합니다.
방법 B (수동) — 실매출액 셀을 직접 클릭하여 숫자를 입력합니다.
💡 주문번호·상품주문번호 컬럼의 앞 8자리(yyyymmdd)도 날짜로 자동 인식합니다 (예: 2026050178033111 → 2026-05-01).
4
저장
💾 저장 버튼을 클릭하면 브라우저 로컬스토리지에 저장됩니다. 새로고침해도 데이터가 유지됩니다. 백업이 필요하면 ⬇ JSON 내보내기로 파일을 저장하세요.
엑셀 파일을 올렸는데 날짜/금액이 인식이 안 될 때
컬럼명에 "결제", "주문", "금액" 키워드가 포함되어 있는지 확인하세요. 날짜의 경우 결제완료일·주문일·주문결제완료일·주문번호(앞8자리) 형태를 인식합니다.
새로고침 시 데이터가 사라질까요?
💾 저장 버튼을 한 번 클릭해두면 브라우저를 닫거나 새로고침해도 데이터가 유지됩니다. 단, 브라우저 캐시를 전체 삭제하면 초기화될 수 있으므로 중요한 데이터는 JSON 내보내기로 백업하세요.

🧮 탭 2 — 예상매출 계산

1
요일지수 설정 (왼쪽 패널)
요일별 판매 비중(%)을 입력합니다. 엑셀 파일 업로드로 과거 데이터를 분석해 자동 계산하거나, 직접 숫자를 입력할 수 있습니다. 채널별로 다른 지수를 설정하려면 상단 드롭다운에서 채널을 선택하세요.
💡 "균등분배" 버튼을 누르면 14.3%씩 동일하게, "정규화"는 합계가 100%가 되도록 자동 조정합니다.
2
월·화·수 실매출 입력 (오른쪽 패널)
이번 주 월요일, 화요일, 수요일의 실제 판매액을 입력합니다. 아래에 합계와 일평균이 자동 계산됩니다.
3
상황 보정 선택
현재 판매 상황에 맞는 버튼을 선택합니다: 평시(×1.0), 행사 예정(살짝 상향), 행사 중(상향), 행사 직후(하향). 슬라이더로 보정 배율을 세밀하게 조정할 수 있습니다.
4
예상매출 확인 및 현황표 적용
계산된 예상 주간 매출이 아래에 표시됩니다. "📋 현황표에 적용" 버튼을 클릭하면 매출 현황 탭의 해당 주차 예상매출액에 자동으로 입력됩니다.
